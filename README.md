# PMDnD 性格测试

发现你在不可思议的宝可梦世界中的角色——一个没有训练家、没有精灵球的世界，每一位宝可梦都**为自己而活**。

## 关于

本测试基于 **PMDnD（Pokémon Mystery Dungeons & Dragons）** 世界观——一个将《宝可梦不可思议迷宫》与《龙与地下城》规则融合的跑团系统。

80 道沉浸式问题将探索你的：

- **宝可梦种族** — 基于种族值 + 体型偏好，从 931 种候选宝可梦中匹配
- **战斗职业** — 12 种 D&D 风格职业，从狂战士到法师
- **九宫格阵营** — 守序/混乱 × 善良/邪恶
- **神祇青睐** — 31 位 PMDnD 世界观神祇，基于概念标签 + 阵营亲密度匹配

## 技术

- 纯静态单页应用，无需服务器
- 精灵图资源来自 [PMDCollab/SpriteCollab](https://sprites.pmdcollab.org/)
- 宝可梦数据来自 [PokéAPI](https://pokeapi.co/)

## 本地运行

直接在浏览器中打开 `index.html`，或使用任意静态文件服务器。
